Importación de porcentajes para Convenio Multilateral

Nombre del archivo Examinar

Se debe indicar el nombre del archivo a importar, puede buscarse mediante el botón Examinar.

Provincia

Código de la provincia cuyos porcentajes desean importarse.

Preproceso

Archivo

Es posible indicar un archivo para preproceso del archivo a importar, con extensión awk, que se aplicará automáticamente sobre el archivo a importar antes de iniciar el proceso de importación. Se dispone de un botón para buscar dicho archivo. Para que este proceso funcione es preciso que el archivo awk.exe se ubique en directorio Programs de la instalación de BASCS. Vale aclarar que AWK es un lenguaje de programación diseñado para procesar datos basados en texto. Esta funcionalidad puede utilizarse en los casos en que el archivo a importar tiene un formato muy especial y debe adecuarse antes de la importación.

Es posible no eliminar, al finalizar el proceso de importación, los archivos intermedios generados al utilizar un archivo de preproceso, o en la eliminación de caracteres extraños del archivo original.

Para ello podrá definirse una variable ambiental de usuario con nombre NOBORRARARCIMP, que en caso de crearse indicando que tiene como valor 1, provoca que el sistema no elimine los archivos mencionados al finalizar el proceso de importación.

Los archivos se podrán encontrar en el directorio de instalación de BASCS o en la ubicación del archivo de importación. El correspondiente a la aplicación del archivo de preproceso tiene el prefijo ~PRE y la extensión TXT. El correspondiente a la eliminación de caracteres extraños tiene el mismo nombre que el archivo de importación y la extensión BAS.

En caso de no existir la variable o no tener valor 1, dichos archivos se eliminarán al finalizar el proceso de importación, tal como ocurría hasta ahora.

Procesar por lotes de registros

Es posible realizar el proceso de importación dividiendo la información a procesar por lotes, indicando la cantidad de registros a procesar por vez, o en forma conjunta. Ya sea que se trabaje por lotes, o no, el proceso no puede interrumpirse hasta que se finalice con el procesamiento de todos los registros. El hacerlo por lotes, en algunos equipos, puede mejorar el uso de los recursos de los mismos.

Coeficientes por defecto %

Porcentaje de alícuota de Convenio Multilateral por defecto.

Generar

Para comenzar la importación se debe presionar el botón Generar.

Para tener en cuenta

Al finalizar el proceso de importación, ya sea por lotes o no, el proceso informa la cantidad de registros procesados, la cantidad de importados, y los registros que no se hayan podido importar por presentar alguna anomalía, ya sea por números de CUIT incorrectos u otros problemas.

Los errores detectados durante el proceso se mostrarán en el detalle de errores y separados en dos grupos, Registros con errores y Registros observados. Los números de CUIT inexistentes o inválidos se mostrarán como Observaciones. Se informará además el número de línea del archivo en la que se detecta cada error u observación.
