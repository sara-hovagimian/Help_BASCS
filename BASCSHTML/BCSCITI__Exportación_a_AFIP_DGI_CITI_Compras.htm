S.I.Ap. AFIP DGI C.I.T.I Compras

A través de este aplicativo es posible generar archivos con toda la información que se requiere para que sea importada mediante el programa de la AFIP C.I.T.I Compras.

Formato

En este campo se ofrece el código del formato estándar previsto por BAS CS para generar el archivo.

Si se desea acceder a las características y al detalle de un formato se debe presionar la tecla F4 estando ubicados en el campo formato que se quiere consultar o definir. Debe tenerse en cuenta que los formatos estándar se pueden modificar pero no eliminar.

Los formatos estándar que ofrece BAS CS pueden modificarse. Para hacerlo se puede consultar:

Formato de archivos de exportación.

Nombre de archivo

Se debe indicar la ruta y el nombre correspondientes al archivo a generar.

Período Año

Se exhibe por defecto, el mes y el año en curso. El período que se indique no puede ser posterior a la fecha del día.

Empresa

Se exhibe la empresa que se ha indicado como empresa por defecto.

Generar

Una vez completados los datos, al presionar este botón, el sistema verifica que no se haya omitido ningún dato y genera el archivo en el directorio que se haya indicado.

En caso que se haya omitido algún dato, o que por algún motivo no se pueda crear el archivo, se emitirá un mensaje al usuario.

El archivo incluye los comprobantes de los tipos mencionados cuya fecha corresponda al período seleccionado, que pertenezcan a la empresa seleccionada, que no estén eliminados y cuyo importe de I.V.A. (suma del I.V.A. inscripto y no inscripto) sea mayor a cero.

Este utilitario se puede ejecutar varias veces sobre la misma empresa o sobre diferentes empresas, para el mismo período o para períodos diferentes, sin necesidad de salir y volver a entrar al mismo.
